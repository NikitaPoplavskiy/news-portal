
## Install
01. `git clone https://github.com/NikitaPoplavskiy/news-portal.git`
02. `cd laravel-news-portal`
03. `composer update`
04. `cp .env.example .env`
05. `php artisan key:generate`
06. `run homestead.sql`
07. `php artisan serve`
